# 《JS 函数的执行时机》
---
1. 函数的调用时机举例
```JavaScript
let i = 0
for(i = 0; i<6; i++){
  setTimeout(()=>{
    console.log(i)
  },0)
}
// 这个代码会打印出6,6,6,6,6,6
```
问:为什么这个代码会打印出6个6?
答:调用时机不同，结果不同。循环执行完后 i=6 之后再打印出 i ，结果就是6个6。

---
2. 让上面代码打印 0、1、2、3、4、5 的方法
```JavaScript
for(let i = 0; i<6; i++){
  setTimeout(()=>{
    console.log(i)
  },0)
}
// 这个代码会打印出0,1,2,3,4,5
```
由于在JS中let与for一起使用会额外加东西。将初始化语句let写入for循环，每次循环都会复制一份 i ，循环结束后会得到6个不同的 i ,循环结束后再打印出6个不同的 i ，结果就是0,1,2,3,4,5。
